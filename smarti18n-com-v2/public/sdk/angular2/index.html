smarti18n

SmartI18N will die Internationalisierung von Webprojekten vereinfachen. Wir bieten einen Online Editor zum Anpassen von I18N Texten in Realtime. Das bedeutet, dass nach einer Anpassung der Texte, die Änderung direkt live verfügbar sind.
