Community

Wie viele Software Projekte profitiert auch SmartI18N von OpenSource Software. Wir wollen offen für neue Ideen sein und daher wird unser Quelltext unter der Apache 2 Lizenz veröffentlicht. Der Code steht bei GitHub zur Verfügung und kann an die persönlichen Anforderungen angepasst werden. Wir hoffen, dass die Community und wir von diesem Konzept gleichermaßen profitieren.
